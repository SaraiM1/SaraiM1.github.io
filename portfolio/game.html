This game is called Hamster Detective, in this game you are a detecive hired to find the hamster in order to return it back to its owner. You use the mouse in order to find thr hamster, there are 3 levels to play.

This game is meant to be fun but was created with the intention to being awarenes to pet care, such as hamster care. Hamsters are commonly one of children's first pet because they seem to be easy and cute to take care of. However, most of the time this ends up in neglecting hamster pets because children are not ready for the responsiblity that comes with any pet. Cute, adorable hamsters are no expection. Below the game there are pet care links in case you are interested in learning about hamster care or if you are interested in getting a hamster.
